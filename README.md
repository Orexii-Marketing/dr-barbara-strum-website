# Dr. Barbara Sturm Affiliate Website

A luxury skincare affiliate marketing website featuring Dr. Barbara Sturm products.

## Deployment

This is a static React application optimized for Vercel deployment.

### Quick Deploy to Vercel

1. Upload project files to GitHub repository
2. Connect repository to Vercel
3. Deploy with default settings - vercel.json handles configuration

### Custom Domain Setup

1. Add custom domain in Vercel dashboard
2. Configure DNS records as provided by Vercel
3. SSL certificate provisions automatically

## Features

- 25+ Dr. Barbara Sturm product pages
- Amazon affiliate link integration
- Newsletter signup functionality
- Search and filtering
- SEO optimized
- Mobile responsive
- Google Ads conversion tracking ready
